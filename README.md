# FightingGame

Two samurais fight it out in an HTML, CSS, Javascript game. Play the game here: https://two-samurai-fighting-game.netlify.app/

Player 1 Keyboard Controls: 
- "A" - Left 
- "D" - Right
- "W" - Jump
- "D" - Attack

Player 2 Keyboard Controls: 
- "Arrow Left" - Left 
- "Arrow Right" - Right
- "Arrow Up" - Jump
- "Arrow Down" - Attack
